## Hacker News "Show HN" Draft

**Title** (≤80 chars):
Show HN: Alpha Skills – AI skills for automated quant factor research

**Text** (first comment):
Hi HN, I built a set of 7 open-source AI skills that turn any coding assistant (Cursor, Windsurf, Claude Code, etc.) into a quantitative factor research workstation.

The core idea: instead of writing the same IC/quintile/backtest boilerplate for every new factor, you describe what you want in natural language and the AI executes the full research pipeline.

The newest addition is alpha-mine — automated factor mining. It systematically generates candidate factors by combining operators (rolling correlation, z-score, rank) with price/volume data, screens them via IC, and presents the top discoveries with an economic intuition score to guard against data mining.

All skills are self-contained Markdown files — no framework, no package to install. They work by teaching the AI how to write and run the necessary Python code on-the-fly.

Supports A-share (China), Hong Kong, and US equities out of the box.

GitHub: https://github.com/VernonOY/alpha-skills

Would love feedback from anyone doing factor research — what other workflows would benefit from this approach?
